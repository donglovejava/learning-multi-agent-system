# AI 学习宠物伴侣 - 系统级桌面宠物

> 一只住在桌面上的像素猫「小橘」，陪伴学生学习，懂他、鼓励他、帮他规划未来。

## 快速启动

```bash
cd desktop-pet
npm install
npm start
```

## 功能

- **透明窗口**：宠物在桌面上自由行走，不挡操作
- **像素动画**：idle / 走路 / 睡觉 / 开心 / 担心 5 种状态
- **对话气泡**：点击宠物弹出对话，接入讯飞星火 LLM
- **拖拽移动**：可以拖拽宠物到桌面任意位置
- **系统托盘**：右键托盘图标控制宠物状态
- **情感感知**：根据对话内容切换宠物表情

## 技术栈

- Electron（透明窗口 + 系统级集成）
- Canvas 像素画引擎（16x16 逻辑网格，4x 放大）
- HTTP 调用后端 API（复用现有学习系统）

## 宠物状态机

```
idle ←→ walk（随机散步）
  ↓       ↑
sleep   happy（做对题/被点击）
  ↓
worry（连续答错/深夜学习）
```

## 与学习系统集成

宠物通过 HTTP 调用后端 `POST /api/v1/chat` 接口，
复用现有的 11 Agent 架构和学习画像数据。

后端需先启动：
```bash
cd ../backend
uvicorn app.main:app --reload
```
